# BA Resource Tracker — Roadmap

Prinsip: **Sedikit fitur tapi dimaksimalkan, daripada banyak fitur tidak terpakai.**

Berikut daftar fitur potensial untuk masa depan, belum diimplementasi:

---

## 🔜 Coming Soon

### Time Tracking
- Member bisa log waktu kerja per project per hari
- Head bisa lihat rekapitulasi waktu per member & per project
- Export timesheet ke CSV/Excel

### Workload Forecast
- Prediksi overload berdasarkan end date project + alokasi saat ini
- Highlight member yang akan over-capacity 2 minggu ke depan
- Saran auto-rebalancing alokasi

---

## 📋 Backlog

### Notification System
- Email/in-app notification saat assignment berubah
- Reminder untuk member yang alokasinya > 100%
- Notifikasi saat semua task selesai

### Dashboard Analytics Enhancement
- Grafik trend utilization per minggu/bulan
- Project timeline (Gantt view sederhana)
- Heatmap skills matrix

### Leave & Availability Calendar
- Member bisa set jadwal cuti/training
- Kapasitas otomatis tereduksi saat cuti
- Calendar view per tim

### Project Templates
- Template project dengan phase & assignment default
- Clone project untuk project serupa

### Audit Trail
- Riwayat lengkap perubahan data (siapa, kapan, apa)
- Rollback perubahan

### Mobile Optimization
- Progressive Web App (PWA) support
- Push notification via mobile

### Integration
- Jira/Trello import project
- Google Calendar sync untuk leave
- Slack/Teams notification

---

> **Catatan**: Roadmap ini bersifat fleksibel dan akan di-update berdasarkan kebutuhan nyata. Fitur hanya diimplementasi ketika benar-benar dibutuhkan.
